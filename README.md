# Blog
A php project blog built using framework laravel

# Package
- Laravel Breeze (auth scaffolding)
- Spatie Query Builder
- Spatie Json Paginate
- Gridjs (datatable js lib)
- Tailwindcss (css)

# Specification
- Laravel 8
- PHP 7.4
- MYSQL 8
- Redis

# Getting Started
- run vendor/bin/sail up -d
